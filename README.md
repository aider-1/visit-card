# Визитка — Бухгалтерское обслуживание для ИП и ООО

Одностраничный сайт-визитка (React + Vite), собирается в статические файлы.

## Запуск

```bash
npm install
npm run dev       # локальная разработка → http://localhost:5173
npm run build     # сборка статики в папку dist/
npm run preview   # предпросмотр собранной статики
```

После `npm run build` в папке **`dist/`** лежат готовые статические файлы
(`index.html` + `assets/`), которые можно залить на любой хостинг.

> При установке зависимостей на этой машине один раз потребовалось разрешить
> install-скрипт esbuild: `npm approve-scripts esbuild` (уже выполнено).

## ⚙️ Где менять контакты и ссылки

Все контакты и ссылки собраны в одном файле: **`src/data.js`**

```js
export const contacts = {
  phone: '+7 (902) 875-53-07',
  phoneHref: 'tel:+79028755307',
  ...
  telegram: 'https://t.me/',   // ← ВСТАВИТЬ реальную ссылку на Telegram
  max: 'https://max.ru/',      // ← ВСТАВИТЬ реальную ссылку на MAX
}
```

Иконки Telegram и MAX в шапке, в блоке «Контакты» и во всплывающем окне
«Получить консультацию» ведут по этим ссылкам. **Сейчас там заглушки** —
замените `telegram` и `max` на настоящие адреса и пересоберите (`npm run build`).

## Структура

```
src/
  App.jsx              — сборка страницы
  data.js              — тексты, контакты, ссылки (единая точка настройки)
  index.css            — все стили, анимации, адаптив
  hooks/useReveal.js   — плавное появление блоков при скролле
  components/
    Header.jsx         — шапка (навигация + бургер-меню на мобилках)
    Hero.jsx           — первый экран
    About.jsx          — «Почему нам доверяют?»
    Services.jsx       — «Услуги» (3 карточки)
    Contacts.jsx       — «Контакты»
    Footer.jsx         — подвал
    ConsultModal.jsx   — окно «Получить консультацию»
    SocialLinks.jsx    — иконки Telegram / MAX
  assets/              — изображения из макета
```
